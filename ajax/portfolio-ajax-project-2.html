Amazon Clone

Project Info:

This Amazon clone truly strengthened my understanding of React and Redux and solidified my confidence in using them. It also gave me much needed practice in CSS to pretty much clone any design/layout I see via code. Not only does it look similar to Amazon's website, I was also able to implement a login & registration with user authentication via firebase, and handle payment processing via stripe and Express.js for a backend server.

Project Details:

  • Technologies: HTML5, CSS, Javascript, React, Redux, Firebase, Stripe, Express.js
